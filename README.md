AML-manager

A professional tool designed to manage and monitor Anti-Money Laundering (AML) compliance for legal and professional firms. It automates expiration tracking based on client risk profiles and provides a visual dashboard for real-time compliance oversight.

Features
- Dynamic Risk Assessment: Automatically calculates the next review date based on client risk (High, Medium, Low).
- Compliance Dashboard: Two-column layout for efficient client management and monitoring.
- Document Checklist: Tracks mandatory documentation for each client.
- Real-time Status: Visual feedback on compliance status (Green/Yellow/Red).
- Data Persistence: Uses LocalStorage to save and retrieve client data locally.
